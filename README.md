A simple bug-loggger application for a team where users can add bug logs and delete them.

### Install

#### Clone this repo

```
git clone https://github.com/bradtraversy/simple-electron-react.git
```

#### Install dependencies

```
npm install
```

or

```
yarn
```

### Usage

#### Run the app

```
npm run start
```

or

```
yarn start
```

#### Build the app

```
npm run package
```

or

```
yarn package
```
